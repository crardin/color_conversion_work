Description:
The MacAdam limits are the boundaries of the set of surface reflectance colours,
when viewed in a given illuminant.  Since the Munsell system classifies all
surface reflectance colours, a colour at the MacAdam limits (also called an optimal
colour) can also be seen as a colour whose Munsell chroma attains a maximum for a
given Munsell hue and value.  This file lists the maximum chroma possible for
colours for all of the 40 standard Munsell hues, and 9 integer Munsell values.
The illuminant is assumed to be the Munsell standard, illuminant C.
This file was generated by the routine MakeMaximumChromaMatrix.m, which
has been contributed to ColorLab, an open source set of colour-related routines
for Matlab or Octave.  A related file is MacAdamLimitsFromRenotationData.txt,
which gives the maximum chroma for which data is available in the Munsell
renotation [Newhall1943, Table I].

[Newhall1943] S. M. Newhall, D. Nickerson, & D. B. Judd, "Final
        Report of the O.S.A. Subcommittee on the Spacing of the Munsell
        Colors," Journal of the Optical Society of America, Vol. 33,
        Issue 7, pp. 385-418, 1943.
