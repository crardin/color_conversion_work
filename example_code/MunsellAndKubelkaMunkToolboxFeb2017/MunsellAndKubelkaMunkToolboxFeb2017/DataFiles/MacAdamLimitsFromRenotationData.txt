Description:
The MacAdam limits are the boundaries of the set of surface reflectance colours,
when viewed in a given illuminant.  The Munsell renotation [Newhall1943] expresses
Munsell colours (which are surface reflectance colours) in xyY coordinates, with
respect to illuminant C.  Table I of [Newhall1943] lists xyY coordinates for
Munsell colours defined by 40 hue designations, 9 integer values, and even chromas.
The MacAdam limit colours occur when, for a given hue and value, the chroma
reaches a physically realizable maximum.  The data in this file lists the maximum
chromas that appear in Table I of [Newhall1943].
This file was generated by the routine MakeRenotationMacAdamLimitMatrix.m, which
has been contributed to ColorLab, an open source set of colour-related routines
for Matlab or Octave.  That routine reads in the renotation data from the file
MunsellRenotation.txt, which is a modified version of the file real.dat, obtained
from [MCSL2010].
The data in this file is expected to be useful when inverting the Munsell renotation,
and when determining what colours are within the MacAdam limits.  For ease of use,
the data has also been saved in the Matlab file MaxChromasInRenotationData.mat,
which can be loaded directly into Matlab or Octave.
Since this file is based on renotation data, every maximum chroma listed is an even
integer.  Another file, MaxChromasForStandardMunsellHuesAndValues.txt, based on
extrapolations of the renotation data, gives more accurate estimates of maximum
chroma, which are not required to be integers.

[MCSL2010] http://www.cis.rit.edu/mcsl/online/munsell.php, as of June 12, 2010.
[Newhall1943] S. M. Newhall, D. Nickerson, & D. B. Judd, "Final
        Report of the O.S.A. Subcommittee on the Spacing of the Munsell
        Colors," Journal of the Optical Society of America, Vol. 33,
        Issue 7, pp. 385-418, 1943.
